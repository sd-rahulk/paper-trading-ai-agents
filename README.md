# SimTrade AI - Premium Dashboard

SimTrade AI is a high-end fintech dashboard built for monitoring autonomous AI trading agents. It features a premium dark theme, glassmorphism aesthetics, and real-time data visualization.

## ✨ Features

- **Live Dashboard**: Real-time portfolio tracking, active positions, and AI confidence metrics.
- **Algorithm Analytics**: Visual breakdown of AI agent weights and strategy performance.
- **Trade History**: Detailed logs of all executed trades with PnL analysis and CSV export.
- **Advanced Settings**: Customizable risk profiles (Safe, Balanced, Aggressive) and capital management.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

## 🛠 Tech Stack
- **Framework**: Vue 3 (Composition API)
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4
- **State Management**: Pinia
- **Routing**: Vue Router
- **Charts**: Chart.js + vue-chartjs
- **Icons**: Lucide Vue Next

## 🎨 Visual Style
- **Theme**: Dark Mode
- **Design System**: Glassmorphism with backdrop blurs and subtle gradients.
- **Animations**: Smooth transitions and interactive chart hover effects.
